I had an interview. The position is MLE and it is a coding interview which related to ML. The question is to implement K-Means algorithm.

So what is K-means? K-means is an algorithm that divide given n points into k clusters.

The process is like this: 1. Initialize k points as the centroid clusters. (There are different ways to do initialization) We can random pick k points to init as the centroids here.

2. Calculate the distances between each point and each centroid. Determine the centroid that each point belongs to by the distance. Make sure there is at least one point which belongs to each cluster.

3. Update the centroids by calculating the new centroid of each cluster.

4. Iterate from step 2 to step 4 until it hits the stopping criteria. The stopping point could be the cluster each point belongs to doesn't change from previous one.
